About VibroJaw

Wearable vibroacoustic therapy and real-time monitoring for temporomandibular joint disorder — for under $30.

The Problem

Temporomandibular joint disorder (TMD) affects an estimated 10 million Americans — disproportionately women and young adults — causing chronic jaw pain, headaches, and restricted movement. Yet the monitoring tools available are either clinically locked or entirely absent from patients' lives between appointments.

Current management relies on passive solutions: expensive custom mouthguards ($200–$600), episodic Botox injections ($300–$500/session), or overnight polysomnography in a sleep lab ($1,000–$3,500). None produce the longitudinal, real-world clenching data that would actually help clinicians manage the condition.

Our Solution

VibroJaw is a lightweight headband wearable with two simultaneous functions: it detects jaw clenching events in real time using a PVDF piezoelectric film sensor, and delivers targeted vibroacoustic biofeedback via vibration motors to interrupt the clenching reflex and relax the masseter and temporalis muscles.

Every event is timestamped and transmitted wirelessly to a Flask web server, then broadcast live to a monitoring dashboard via Socket.IO — creating the first accessible longitudinal clenching health record for TMD patients.

How It Works

  • Detect: PVDF piezo film on the temporalis muscle reads jaw vibration at 200 Hz. A rolling-average filter removes electrical noise, and a state machine distinguishes valid clenches from brief artifacts using a configurable duration threshold.
  • Respond: On a valid clench, the ESP32 fires a PWM-controlled vibration motor at 50–100 Hz. Intensity and duration are configurable from the web dashboard, pushed wirelessly to the device.
  • Transmit: Each event is immediately POSTed to the Flask server via HTTP. A 5-second batch summary is also sent periodically for trend data, even during quiet periods.
  • Display: The server broadcasts new data via Socket.IO to any connected browser. The chart and event log update live — no page refresh needed.
  • Adapt: The ESP32 polls the server for config updates every 10 seconds. Threshold or vibration changes made on the dashboard apply to the device automatically — no reflashing required.

Hardware (Total BOM: ~$23)

  • ESP32 S3 DevKitC (~$5): Built-in WiFi and Bluetooth. Handles ADC sensing, rolling-average filter, state machine, motor PWM, HTTP, and config polling.
  • PVDF Piezoelectric Film Sensor ($7.35): Flexible polymer sensor that generates voltage proportional to mechanical deformation. Validated in peer-reviewed bruxism detection literature for external temporalis placement.
  • DC Vibration Motors — 10 pack ($5.99): 3V coin vibration motors, 12,000 RPM. Used for biofeedback delivery at therapeutic frequencies.
  • Adjustable Strap ($4.99): Elastic headband for comfortable external placement over the temporalis muscle.

Scientific Foundation

  • Calvo-Lobo et al. (2020, PMC) — Randomized double-blind placebo-controlled trial showing significant TMJ pain reduction using local vibration at 50–100 Hz on masseter and temporal muscles.
  • Journal of Prosthodontic Research (2023) — 14-week trial validating piezo-detected bruxism + vibration biofeedback — the same closed-loop architecture as VibroJaw, applied externally.
  • US Patent US10820853B2 (2020) — Validates temporalis muscle as optimal external sensor placement for clenching detection with minimal false positives from speech or facial movement.
  • PMC / Clinical and Experimental Dental Research (2024) — Systematic review identifying the precise gap in real-world external wearable bruxism monitoring that VibroJaw addresses.
